if/ else / else if alplicados al Dom

Condicionales en el DOM

Los condicionales permiten que el DOM reaccione diferente según los datos del usuario. Combinamos if / else if / else con document.getElementById para cambiar contenido y estilos dinámicamente. 

// ── Leemos el valor del input ─────────────────────────────
const edad = Number(document.getElementById("inputEdad").value);

// ── Evaluamos con if / else if / else ─────────────────────
if (edad >= 18) {
  // Modificamos el DOM si la condición es verdadera
  document.getElementById("resultado").textContent = "Mayor de edad ✅";
  document.getElementById("resultado").style.color = "#5ACCA4"; // verde
} else if (edad >= 12) {
  document.getElementById("resultado").textContent = "Adolescente 🔵";
} else {
  document.getElementById("resultado").textContent = "Niño/a 🟡";
}

// ── Cambiar clases CSS dinámicamente ─────────────────────
// classList.add()    → agrega una clase
// classList.remove() → elimina una clase
// classList.toggle() → alterna una clase (si está la quita, si no la pone)
elemento.classList.add("activo");
elemento.classList.remove("inactivo");
