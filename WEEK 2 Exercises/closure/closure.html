Recuerda: ¿qué es un Closure?

Un closure ocurre cuando una función interna recuerda las variables de su función exterior, aunque esa función ya haya terminado de ejecutarse. La función retornada lleva consigo una "mochila" con esas variables. 

// Ejemplo base de closure
function crearMensaje(prefijo) {
  // "prefijo" quedará en el closure de la función retornada
  return function(texto) {
    return prefijo + " " + texto; // accede a "prefijo" del exterior
  };
}

const avisar  = crearMensaje("⚠️ AVISO:");
const informar = crearMensaje("ℹ️ INFO:");

avisar("El sistema caerá")   // "⚠️ AVISO: El sistema caerá"
informar("Todo está bien")   // "ℹ️ INFO: Todo está bien"
