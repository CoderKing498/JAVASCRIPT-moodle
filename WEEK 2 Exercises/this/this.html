Recuerda: cómo funciona "this"

Dentro de un método de objeto, this apunta al propio objeto. Lo usamos cuando un método necesita acceder a las propiedades del objeto donde vive. 

const alumno = {
  nombre: "Ana",
  notas : [80, 90, 75],

  promedio() {
    // "this" apunta al objeto "alumno"
    const suma = this.notas.reduce((acc, n) => acc + n, 0);
    return suma / this.notas.length;
    // this.notas.length → accede al array del propio objeto
  },

  presentar() {
    // this.nombre → accede a la propiedad del propio objeto
    // this.promedio() → llama al método del propio objeto
    return `${this.nombre} tiene promedio ${this.promedio()}`;
  },
};

alumno.presentar(); // "Ana tiene promedio 81.67"
